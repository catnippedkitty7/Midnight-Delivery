Midnight Delivery v3.0 Supabase Leaderboards

Setup:
1. Create a Supabase project.
2. Open SQL Editor and run supabase_setup.sql.
3. Open index.html.
4. Find:
   const SUPABASE_URL = 'PASTE_YOUR_SUPABASE_PROJECT_URL_HERE';
   const SUPABASE_ANON_KEY = 'PASTE_YOUR_SUPABASE_ANON_KEY_HERE';
5. Paste your Project URL and anon/public key.
6. Host the folder on Netlify, itch.io, GitHub Pages, or similar.

Never paste your service_role key into the game.
